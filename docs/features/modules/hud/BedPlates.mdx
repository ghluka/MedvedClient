# BedPlates

Displays what blocks are defending a bed, letting you either find the weakest bed defenses, or plan what tools you need to bring. Useful for the Bed Wars minigame.

:::warning
This module has either been added or heavily updated in the experimental version,
do not expect configuration options or this module to be available if you use the
latest release.
:::

---

### Search Range

How far away (in blocks) to scan for beds.

### Shell Radius

How many blocks out from the bed to check for defense blocks.

### Scan Interval

How frequently (in milliseconds) to re-scan for beds and their defenses. Lower values are more responsive but cost more performance.

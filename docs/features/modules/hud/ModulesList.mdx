# Modules List

Shows all enabled modules

:::warning
This module has either been added or heavily updated in the experimental version,
do not expect configuration options or this module to be available if you use the
latest release.
:::

---

### Show visuals

Show modules on the list that only affect visuals, such as ESP.

### Left border

Draw a border on the left side of the module list.

### Background

Render a background behind the module list.

### BG color

Background color for the module list, with alpha support.

### Text shadow

Draw a shadow behind the module text.

### Sort

How enabled modules are ordered.

- **SIZE**: sort by width or size
- **ALPHA**: sort alphabetically
- **ORDER**: preserve module order

### Reverse order

Flip the order of the sorted module list.

### Capitalize

Text casing for module names.

- **UPPER**: uppercase text
- **LOWER**: lowercase text
- **TITLE**: keep original casing

### Shadow

Render a shadow around the module names.

- **NONE**: no shadow
- **SIMPLE**: just a dropshadow on the bottom right
- **SOFT**: a shadow that covers all sides of the text

#### Shadow strength

How opaque the shadow will be

### Pad x

Horizontal padding in pixels around each module entry.

### Pad y

Vertical padding in pixels around each module entry.

### Show config

Toggle display of module configuration lines in the list.

# Flight <sup><Chip label="Warning" color="warning" size="small" component="a" href="/MedvedClient/guides/anticheat-safety#risk-labels" icon={<WarningIcon />}/></sup>

Allows you to fly or glide in the air

---

### Mode

Choose the flight mode.

- **VANILLA**: normal flight behavior
- **CREATIVE**: creative-like flight
- **AIR HOP**: hop through the air
- **GLIDE**: glide while descending

### Speed

Flight speed multiplier in vanilla mode.

### Anti-kick

Prevent server kick while flying.

### Anti-kick mode

Choose the anti-kick timing method.

- **LATEST**: use the latest anti-kick method
- **LEGACY**: use the legacy anti-kick method

### Interval

Tick interval used by the latest anti-kick method.

### Distance

Distance value used by the latest anti-kick method.

### Old interval

Interval used by the legacy anti-kick method.

# Auto Totem

Automatically equips totems of undying to your offhand slot

---

### Open Inventory

Allow the module to open your inventory to equip totems. If disabled,
it'll only equip totems if you manually open your inventory.

### Random Slot

Choose a random totem slot when equipping.

### Delay (ms)

Delay between each automatic totem equip action.

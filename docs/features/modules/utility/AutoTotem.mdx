# Auto Totem

Automatically equips totems of undying to your offhand slot

---

### Open Inventory

Allow the module to open your inventory to equip totems.

### Silent Open

Open the inventory without visible screen interruption when using inventory mode.

### Inventory Only

Only run while the inventory screen is open.

### Random Slot

Choose a random totem slot when equipping.

### Delay (ms)

Delay between each automatic totem equip action.

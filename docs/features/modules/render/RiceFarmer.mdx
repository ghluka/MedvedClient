# Rice Farmer

Renders a conical hat above your player

---

### Color Mode

Sets the hat's colour

- **RAINBOW**: makes the hat have a rainbow gradient
- **GRADIENT**: lets you set the hat to a gradient between two colours
- **SINGLE**: lets you set the hat to a single colour
- **THEME**: makes the hat have the same colour as your UI theme

### Radius

Sets the radius of the cone

### Height

Sets the height of the cone

### Position

How much the cone is offset on the y axis

### Rotation

How fast the hat spins

### Angles

How detailed the cone is.
At 4 it will appear like a square pyramid, at 90 it will appear like a circular cone.

### First Person

Is the hat visible when you look up in first person?

### Shade

Toggles whether the apex of the cone is slightly more opaque (0.8) than the rim, when disabled they have the same transparency (0.5)

# Nametags

Replaces vanilla player nametags with custom screen-space nametags.

:::warning
This module has either been added or heavily updated in the experimental version,
do not expect configuration options or this module to be available if you use the
latest release.
:::

---

### Mode

Selects the nametag design.

- **CLEAN**: compact text tag with optional health and distance
- **EXPLICIT B9**: ported version of Explicit B9's nametags

### Max Distance

The farthest distance a player can be before their custom nametag stops rendering.

### Scale

Scales the nametag text and layout.

### Y Offset

Moves the nametag higher or lower above the player's hitbox.

### Health

Shows the player's current health.

### Distance

Shows the player's distance from you.

### Health Mode

- **HEARTS**: displays health in hearts
- **PERCENTAGE**: displays health as a percentage-style value

### Armor

Only visible for modes that support it.
Draws the target's held item and armor above the nametag.

### Durability

Only visible for modes that support it.
Shows remaining durability above damageable equipment.

### Enchantments

Only visible for modes that support it.
Shows compact enchant labels generated from the enchantment display names.

### Background

Toggles the nametag background.

### BG Color

Background colour and alpha.

### Text Color

Main text colour.

### Text Shadow

Toggles text shadow.

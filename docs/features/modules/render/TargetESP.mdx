# Target ESP

Draws a moving halo around your current combat target.

:::warning
This module has either been added or heavily updated in the experimental version,
do not expect configuration options or this module to be available if you use the
latest release.
:::

---

### Color

The halo colour and alpha.

### Ignore Target Filter

When enabled, Target ESP can render recent or module-selected targets even if
they would normally be rejected by the global **Target Filter** rules.

### Speed

Controls how quickly the halo travels from feet to head and back down.

### Line Width

Base thickness of the halo ring.

### Expand

Expands the halo outward from the target's hitbox.

### Glow Rings

Controls how many extra soft rings are drawn around the main halo.
Higher values make the effect feel more like a glow.

### Linger MS

How long Target ESP stays visible after a direct attack.

### Smoothness

Number of line segments used to draw the halo.
Higher values make the halo rounder.

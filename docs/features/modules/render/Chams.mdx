# Chams

Renders player models, armor, and held items through walls.

:::warning
This module has either been added or heavily updated in the experimental version,
do not expect configuration options or this module to be available if you use the
latest release.
:::

---

### Style

Controls the material used on the player model.

- **SHADER**: replaces the player's material with a custom shader
- **NONE**: keeps the normal player textures while removing world depth

### Shader

Only visible in **SHADER** style. Selects the shader material used for players,
armor, and held items.

- GALAXY
- VOID

# Animations

Affects item animations and how you hold items in first-person view

---

### Mode

Selects the sword blocking animation style to use.

- **ONE SEVEN**
- **PUSHDOWN**
- **SPIN**
- **Chill**
- **HELICOPTER**

### Translate Y

Adjusts the vertical height of the item while blocking.

### Swing Scale

Adjusts the scale and intensity of the swinging motion while blocking.

### Swing Duration

Changes the speed of the item swing animation in ticks. Lower values produce a faster swing.

### Main Hand & Off Hand Overrides

Enabling **Main Hand** or **Off Hand** unlocks advanced transformation sliders to manually configure how hands render on your screen.

- **Scale**: Changes the size of the held item.
- **X / Y**: Changes the horizontal and vertical screen offsets.
- **Rot X / Y / Z**: Pitches, yaws, and rolls the angles of the item.

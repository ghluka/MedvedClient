# Animations

Affects item animations and how you hold items in first-person view

:::warning
This module has either been added or heavily updated in the experimental version,
do not expect configuration options or this module to be available if you use the
latest release.
:::

---

### Mode

Selects the sword blocking animation style to use.

- **ONE SEVEN**
- **PUSHDOWN**
- **SPIN**
- **Chill**
- **HELICOPTER**

### Visual block

If you swing any item that can be used to block attacks, it will display the
block hit animation regardless of if you are blocking.

#### Visual block swords

Makes it so any sword, regardless of if it can block attacks, will show the
block hit animation.

### Translate Y

Adjusts the vertical height of the item while blocking.

### Swing Scale

Adjusts the scale and intensity of the swinging motion while blocking.

### Swing Duration

Changes the speed of the item swing animation in ticks. Lower values produce a faster swing.

### Main Hand & Off Hand Overrides

Enabling **Main Hand** or **Off Hand** unlocks advanced transformation sliders to manually configure how hands render on your screen.

- **Scale**: Changes the size of the held item.
- **X / Y**: Changes the horizontal and vertical screen offsets.
- **Rot X / Y / Z**: Pitches, yaws, and rolls the angles of the item.

### Cancel equip progress

When you equip an item, it shows an animation of it rising into your hand, this
removes this animation to make your hotbar feel more responsive.

# Client Brand

Spoofs the client brand sent to the server on join

---

### Brand

Choose the client brand string to send to the server.

- **VANILLA**: vanilla
- **LUNAR**: lunarclient
- **FORGE**: forge
- **FABRIC**: fabric
- **OPTIFINE**: optifine
- **FEATHER**: feather
- **CUSTOM**: use a custom brand string

### Custom brand

Custom brand string used when Brand is set to CUSTOM.

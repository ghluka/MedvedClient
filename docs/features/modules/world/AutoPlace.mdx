# Auto Place

Automatically places blocks when looking at a block surface

:::warning
This module has either been added or heavily updated in the experimental version,
do not expect configuration options or this module to be available if you use the
latest release.
:::

---

### Require holding

Only place blocks when the use key is held.

### Only air

Only place blocks against air surfaces.

### Block Whitelist

Blocks to allow placing. Defaults to wool only.

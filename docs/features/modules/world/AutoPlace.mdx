# Auto Place <sup><Chip label="Ghost" color="success" size="small" component="a" href="/MedvedClient/guides/anticheat-safety#risk-labels" icon={<VerifiedUserIcon />}/></sup>

Automatically places blocks when looking at a block surface

---

### CPS

Clicks per second while placing blocks.

### Require holding

Only place blocks when the use key is held.

### Only air

Only place blocks against air surfaces.

### Block Whitelist

Blocks to allow placing. Defaults to wool only.

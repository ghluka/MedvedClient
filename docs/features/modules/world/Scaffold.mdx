# Scaffold

Automatically places blocks under you while walking

:::warning
This module has either been added or heavily updated in the experimental version,
do not expect configuration options or this module to be available if you use the
latest release.
:::

---

### Mode

Choose the scaffold placing mode.

- **NINJA**: crouches at the edge of blocks before bridging
- **BREEZILY**: alternates strafing left/right while bridging
- **TELLY**: jumps and then clutches to bridge super fast

### Crouch delay

Delay in milliseconds before auto-crouching when using **NINJA** mode.

### Autoclick CPS

Clicks per second used for automatic block placement, does not affect
bridging consistency.

### Disable on death

Disable scaffold automatically on death.

### Disable on world change

Disable scaffold automatically when changing worlds.

### Block Whitelist

Blocks to use for scaffolding. Defaults to wool only.

# Bed Breaker

Automatically breaks enemy beds

:::warning
This module has either been added or heavily updated in the experimental version,
do not expect configuration options or this module to be available if you use the
latest release.
:::

---

### Mode

Choose the breaking style.

- **LEGIT**: break beds with normal range and rotation
- **BLATANT**: breaks beds through walls, ignoring the defense
- **HYPIXEL**: breaks beds through walls designed for Hypixel

### Rotations

Choose which rotation method to use while breaking.

- **NONE**: no rotation override
- **CLIENT**: client-side rotation override
- **SERVER**: server-side rotation override

### Range

Bed break range used in LEGIT mode.

### Blatant range

Extended range used in BLATANT mode.

### Auto tool

Automatically switch to the best tool when breaking beds.

# Clutch

Bridges blocks back to safety when knocked off an edge

---

### Trigger

Choose when clutching should activate.

- **ALWAYS**: always attempt a clutch
- **ON VOID**: clutch only when over void
- **ON LETHAL FALL**: clutch when falling to a lethal height
- **FALL DISTANCE**: clutch based on fall distance

### Blocks

Minimum fall distance in blocks for FALL DISTANCE mode.

### Return to slot

Return to the previous hotbar slot after clutching.

### Telly

If enabled, clears the rotation when you land on a block so you can get a sprint jump.

### Max blocks

Maximum blocks placed during clutch.

### Rotation speed

How quickly the module rotates while clutching.

### Block Whitelist

Blocks to use for clutching. Defaults to wool only.

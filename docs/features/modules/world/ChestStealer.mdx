# Chest Stealer

Silently steals filtered items from open chests

:::warning
This module has either been added or heavily updated in the experimental version,
do not expect configuration options or this module to be available if you use the
latest release.
:::

---

### Steal Filter

List of items or categories that would be whitelisted to steal.
Leave empty if you want to steal everything.

### Delay (ms)

Delay between each refill action.

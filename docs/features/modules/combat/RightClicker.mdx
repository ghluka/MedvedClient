# Right Clicker <sup><Chip label="Ghost" color="success" size="small" component="a" href="/MedvedClient/guides/anticheat-safety#risk-labels" icon={<VerifiedUserIcon />}/></sup>

Automatically right-clicks while holding the use key

---

### CPS

Clicks per second while holding use.

### Jitter

Add small aim variation after each click.

### Blocks only

Only click when looking at a block.

### Wool only

Only click when looking at wool blocks while blocks only is enabled.

### Allow bow

Allow use-clicks with bows and crossbows.

### Allow eat

Allow use-clicks when consuming food.

### Allow shield

Allow use-clicks while holding a shield.

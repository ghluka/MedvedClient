# Silent Aura

Silently aims and attacks enemies when they enter your range

---

### Targeting

How the module selects and engages targets.

- **Targeting**: choose single target or switch targets
- **Range**: maximum reach in blocks
- **FOV**: field of view for automatic targeting
- **Players only**: restrict attacks to players
- **Auto weapon**: automatically switch to the best weapon when a target is found
- **Auto block**: automatically hold block or sword block after attacking

### Mode

Which attack pacing method Silent Aura uses.

- **CPS**: clicks per second mode
- **SEQUENTIAL**: attack in fixed timed intervals

### CPS

Clicks per second when in CPS mode.

### Extra delay

Additional tick delay when using SEQUENTIAL mode.

### Smoothness

How quickly the aim rotates toward the target in non-APS modes.

# Block Hit <sup><Chip label="Ghost" color="success" size="small" component="a" href="/MedvedClient/guides/anticheat-safety#risk-labels" icon={<VerifiedUserIcon />}/></sup>

Holds block with your sword to reduce incoming damage

---

### Mode

Choose between constant blocking or only when enemies are nearby.

- **STATIC**: always hold block while a valid weapon is equipped
- **DYNAMIC**: only block when an enemy is within range

### Right click only

Only start blocking when the use key is held.

### Release on swing

Briefly stop blocking after an attack to allow swings to register.

### Release (ms)

How long to stop blocking after a swing.

### Dynamic range

Range in blocks used to detect enemies in DYNAMIC mode.

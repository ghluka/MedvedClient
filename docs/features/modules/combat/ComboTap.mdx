# Combo Tap <sup><Chip label="Ghost" color="success" size="small" component="a" href="/MedvedClient/guides/anticheat-safety#risk-labels" icon={<VerifiedUserIcon />}/></sup>

Taps movement keys on attack to adjust velocity for better combos

:::warning
This module has either been added or heavily updated in the experimental version,
do not expect configuration options or this module to be available if you use the
latest release.
:::

---

### Method

The type of tap to perform on attack.

- **W TAP**: briefly stop forward movement and sprint
- **S TAP**: apply a backward tap
- **SHIFT TAP**: force sneaking for a short burst
- **SPRINT RESET**: reset sprint for better knockback control
- **JUMP RESET**: jump when on ground to reset movement

### Tap duration (ms)

How long the tap effect lasts.

### Cooldown (ms)

Delay before the module can tap again.

### Stop sprint

Only used with W TAP to disable sprint while tapping.

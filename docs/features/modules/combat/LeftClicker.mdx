# Left Clicker

Automatically left-clicks while holding the attack key

---

### CPS

How many clicks per second the module should perform.

### Jitter

Adds small random aim movement after each click.

### Only when targeting

Only click when your crosshair is on a valid entity.

### Break blocks

Allow clicking through blocks while active.

### Weapon only

Only auto-click when holding a weapon.

# Keep Sprint <sup><Chip label="Warning" color="warning" size="small" component="a" href="/MedvedClient/guides/anticheat-safety#risk-labels" icon={<WarningIcon />}/></sup>

Retains sprint momentum when attacking entities.

:::warning
This module has either been added or heavily updated in the experimental version,
do not expect configuration options or this module to be available if you use the
latest release.
:::

---

### Retain factor

The multiplier applied to your movement speed when hitting an entity. A value of 100% means you don't lose any momentum.

### Reset sprint

Whether to cancel the visual sprint state upon attacking.

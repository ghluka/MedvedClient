# Hit Select

Restricts auto-clicks and attacks to the most effective moments

---

### Mode

PAUSE keeps only good attacks, ACTIVE cancels weak ones.

### Preference

The priority used to decide attack timing.

- **CRIT**: prefer airborne critical hit windows
- **COMBO**: prefer moving combo hits after hurtstun
- **TIMER**: rely on delay timing only

### Delay (ms)

Minimum wait after a canceled attack before the next one.

### Filter %

Chance to allow an attack even when conditions are not ideal.

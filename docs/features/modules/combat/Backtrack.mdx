# Backtrack

Delays enemy position updates to hit players at their previous position

---

### Mode

Select between manual and lag-based backtracking.

- **MANUAL**: always delay enemy updates by a fixed time
- **LAG**: simulate latency spikes using a lag window

### Delay (ms)

Fixed delay amount for MANUAL mode.

### Lag window (ms)

Delay range for LAG mode.

### Only players

Only backtrack player entities.

### Show ESP

Render a preview of frozen and real entity positions.

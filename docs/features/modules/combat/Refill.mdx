# Refill

Automatically refills your hotbar with healing items

---

### Type

Choose whether to refill pots, soup, or both.

### Mode

Run only in the inventory screen or always.

### Vertical

Prefer vertical refill ordering from the inventory.

### Scatter

Randomize the source slot selection order.

### Delay (ms)

Delay between each refill action.

### Hotbar Clear

Remove junk items from the hotbar before refilling.

### Non Junk Items

Comma-separated list of items that should not be treated as junk when hotbar clearing.

# Rotations

Global aim interpolation settings for all rotation modules

---

### Aim style

Choose the interpolation style for rotation smoothing.

- **LINEAR**: constant interpolation
- **QUADRATIC**: quadratic ease
- **LOGARITHMIC**: logarithmic ease
- **EXPONENTIAL**: exponential ease
- **CIRCULAR**: circular ease

### Speed

Interpolation speed range.

### Max counts

Maximum interpolation count values.

### Max speed deg

Maximum rotation speed in degrees.

### Count jitter

Add noise to the interpolation count.

### Overshoot chance %

Chance to overshoot the target rotation.

### Overshoot amount

Max overshoot amount when overshooting.

### Micro jitter

Small random rotation jitter.

# Installation

:::info
This mod was made for Minecraft version <ReleaseVersion/>, using it on any other version will not work.
:::

## Downloading the latest release

Go onto the project's [latest release page](https://github.com/ghluka/MedvedClient/releases/latest) and download `medved.jar`.

![Download the medved.jar](./img/jar-file.png)

Once you've downloaded the `medved.jar`, place it in your mods folder:

![All required mods installed](./img/three-mods.png)

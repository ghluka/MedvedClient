# What is Grizzly Client?

Grizzly is a Fabric mod for Minecraft that provides an unfair advantage.
It offers ghost modules, which are focused on PvP, and designed to be difficult
to detect by anticheats. It also features a variety of features designed for
minigames.

## What do you get?

### Modules

Being a Fabric mod for the latest version of Minecraft, we have the unique advantage
of being able to give you modules designed for the new style of combat and the old
1.8.9 combat. Depending on your configuration, you can set these modules up to be
obvious but strong enough to win against other cheaters, or unnoticeable advantages
that are designed to be undetected by anticheats.
- [Explore the combat modules.](/category/combat)

We also offer other modules for different scenarios such as for movement,
minigames, and more.
- [Explore all modules.](/category/modules)
